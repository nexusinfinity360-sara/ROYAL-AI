# ROYAL AI - Prompt Engine 



Version: 1.0



---



# Purpose



The Prompt Engine is responsible for understanding user requests, analyzing intent, planning execution, and generating optimized prompts for AI models.



It acts as the communication bridge between the user and the AI Brain.



---



# Responsibilities



- Understand user intent

- Detect project context

- Improve user prompts

- Select the best AI model

- Generate optimized system prompts

- Pass structured instructions to AI agents

# Prompt Optimization



The Prompt Engine should automatically:



- Improve vague prompts

- Add missing project context

- Include coding standards

- Include architecture rules

- Include previous project memory



---



# AI Model Selection



The Prompt Engine selects models based on task type.



Examples:



- Complex Reasoning → Claude

- Coding → GPT / Claude

- UI Design → GPT

- Research → Gemini

- Fast Tasks → DeepSeek
